# SCMALL HK — Chill Zone 香港智能贩卖机

**线上 Landing Page**：https://traineemage.github.io/SCMALL-HK/

---

## 项目定位

Chill Zone 智能贩卖机品牌官网，为线下 **流量入口** 的线上展示窗口，面向香港市场招商加盟合作伙伴。

**核心价值：** 为有意向的商户、加盟商、场地合作方提供品牌信息、合作模式及收益测算的完整入口。

---

## 品牌调性 — 餸潮植根社区

- **社区足迹：** Chill Zone 深入香港各区（启航1331背包客旅舍、嘉多娜面包店、观塘海滨等），强调与本地社区的深度绑定
- **生活方式：** "健康生活，从每一口新鲜开始" — 传递健康、轻负担的品牌形象
- **情感连接：** 贩卖机不只是零售终端，是社区的温度角落

---

## 商业覆盖 — 投资回报空间

| 场景 | 覆盖范围 |
|------|---------|
| 写字楼 | 182+ 精选线路 |
| 校园 | 食堂、图书馆等 |
| 交通枢纽 | 地铁站、高铁站 |
| 社区 | 医院、社区中心 |

**招商模式：** 标准加盟 / 区域代理 / 联营合作，门槛灵活，方案可定制

---

## 运营优势 — 低运营成本

- **全天候自动化销售**，夜间无人工成本
- **本地运维团队**定期补货、清洁、故障响应
- **远程告警系统**接入，24小时在线监控
- **后台实时查看**销售数据与库存状态
